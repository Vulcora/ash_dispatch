    mix compile.ash_dispatch (AshDispatch v0.1.0)

    View Source

    Mix compiler that auto-discovers and compiles all AshDispatch templates.

    This compiler runs during mix compile and scans for templates in two locations:

    1. Convention-based (inline DSL events):

      • Path: lib/{app}/{domain}/templates/{resource}/{event}/
      • Example: lib/magasin/requests/templates/reseller_request/new/email.html.heex

    2. Module-based (event modules):

      • Path: lib/{app}/{domain}/events/{event}/templates/
      • Example: lib/magasin/accounts/events/invited/templates/email.html.heex

    All discovered templates are compiled into AshDispatch.CompiledTemplates module for production use. Development mode uses file-based template loading for fast iteration.

    Usage

    Add to your mix.exs:

    def project do
  [
    compilers: Mix.compilers() ++ [:ash_dispatch]
  ]
end

    The compiler will automatically run during mix compile and generate the compiled templates module.
